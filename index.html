InstantWP – Bootstrap WordPress Theme





InstantWP is a minimal WordPress theme that still has some great features. The theme is perfect for those looking to display portfolio items and also manage a nice WordPress blog. Some of the built-in features include:

  • theme options panel
  • drag and drop homepage builder
  • Font Awesome 4.1.0
  • responsive design
  • and much more including all the Bootstrap goodies

Check it out and let us know if you have any questions.
